# System Specification Document:
## Introduction
Project Title: SimpleSpend: Personal Expense Tracker

## Domain: 
Personal Finance & Wealth Management. 
This domain involves the digital organization of an individual's financial data, 
focusing on transaction tracking, budget adherence, and expense classification.

## Problem Statement: 
Many individuals struggle to maintain a clear overview of 
their discretionary spending, leading to budget overruns. Existing commercial 
apps are often over-complicated, require sensitive bank logins, or include intrusive 
advertisements. SimpleSpend provides a private, manual-entry solution for focused 
financial tracking.

## Individual Scope:
This project is feasible for a single developer because it 
focuses on core CRUD (Create, Read, Update, Delete) functionality. The scope
includes a simple data entry form, a persistent storage layer (SQLite or JSON),
and a basic dashboard to calculate totals by category.
